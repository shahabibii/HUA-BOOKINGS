HUA BOOKINGS

Upload one or more PDFs. The file name should include the event date and name, for example: 2026-03-15-Spring-Gala.pdf or 03-15-2026_Beach-Cleanup.pdf Use Clear before re-uploading if you see duplicate events on the same day.

    Listing-style exports (e.g. RES_LISTING_DETAIL) should include: LEAD_ID, PROPERTY_DESC, ARRIVAL_DATE, NIGHTS, RESV_TYPE. Guest column: we use GUEST_NAME2 when present; otherwise GUEST_NAME (most rows only have the latter). Only RESV_TYPE values FIX, CLB, FCC, HGD, MAX, DRM, DFC, OW, OWU, DXT, OPD, DXP, MGV are loaded. Clear removes loaded rows so you can import a fresh CSV.

    Paste the booking form

    Outlook on the web opened with subject HUA BOOKING and plain-text booking details. Paste for the formatted black-and-white table.

    1. Click inside the email body in Outlook.
    2. Press Cmd+V (Mac) or Ctrl+V (Windows) to paste the form table.
    3. Known booking details from your calendar selection are filled in on the right — complete any empty rows.

    Using the Outlook desktop app? Open the downloaded .eml file from Downloads — it includes the full green/yellow Excel-style template.
