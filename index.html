HUA BOOKINGS

Hosted flyers load for everyone from the flyers/ folder in the repo (see flyers/README.md) — drop a month folder like flyers/june/ with files named 06.01 Spring Gala.pdf or 6.01 Spring Gala.pdf, then push. To add flyers only in this browser, choose files or an entire folder below. Clear removes local uploads only.

    Listing-style exports (e.g. RES_LISTING_DETAIL) should include: LEAD_ID, PROPERTY_DESC, ARRIVAL_DATE, NIGHTS, RESV_TYPE. Guest column: we use GUEST_NAME2 when present; otherwise GUEST_NAME (most rows only have the latter). Only RESV_TYPE values FIX, CLB, FCC, HGD, MAX, DRM, DFC, OW, OWU, DXT, OPD, DXP, MGV are loaded. Clear removes loaded rows so you can import a fresh CSV.

    Send HUA booking

    Choose how you want to open the booking email. Subject will be HUA BOOKING.

    Paste the booking form

    Outlook on the web opened with subject HUA BOOKING. The green/yellow booking form is on your clipboard — paste it into the body.

    1. Click inside the empty email body in Outlook.
    2. Press Cmd+V (Mac) or Ctrl+V (Windows) to paste the formatted form table.
    3. Known booking details from your calendar selection appear after each label — complete any empty rows.

    Opening Outlook desktop

    Outlook desktop should open with subject HUA BOOKING. A formatted .eml file was also downloaded.

    1. If a compose window did not appear, open the downloaded .eml file from Downloads.
    2. The .eml includes the full green/yellow Excel-style template.
